# Setup Requirements
1. OS: Windows (FIFA 18)
2. Pytesseract
3. Tensorflow
4. Keras
5. MatplotLib

# To run
Go to the root directory and run main.py in train or test mode.
Toggle the train_mode variable to run in train or test mode.
